# Приложение: Сравнение двух документов (HTML-отчет)

Простое GUI-приложение на Python (Tkinter) для сравнения двух документов `.txt` или `.docx` и генерации HTML-отчета, где отличия подсвечены красным цветом.

## Запуск

1) Установите зависимости:

```bash
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
```

2) Запустите приложение:

```bash
python app.py
```

3) В интерфейсе выберите два файла и нажмите «Сгенерировать HTML отчет». Укажите путь сохранения и откройте файл в браузере.

## Примечания
- `.txt` читается как UTF-8.
- `.docx` читается с помощью `python-docx`: извлекаются параграфы без форматирования.
- В отчете две колонки: «Документ A» (удаления зачеркнуты красным) и «Документ B» (добавления подсвечены красным).
